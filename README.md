# Análisis de Clientes RFM - AdventureWorks (PowerBI)
Análisis de clientes RFM con PowerBI + SQL usando Base de Datos AdventureWorks de Microsoft

## Descripción del proyecto
Este proyecto presenta un análisis de clientes de la base de datos [**AdventureWork2019**](https://learn.microsoft.com/es-es/sql/samples/adventureworks-install-configure?view=sql-server-ver17&tabs=ssms)  (Venta de bicicletas y accesorios) de Microsoft.


## Objetivos del proyecto
- Identificar los siguientes tipos de clientes:
    - Clientes VIP
    - Clientes Frecuentes
    - Clientes Nuevos
    - Clientes en Riesgo / Inactivos

## Stack Tecnológico
- SQL Server
- PowerBI
- DAX
- Phyton (Pandas)
- Git/GitHub
